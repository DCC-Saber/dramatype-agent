# 剧格 AI DramaType 安全审核规则

## 禁止项

### 1. 禁止 MBTI

- 不能出现 MBTI 四字母标签
- 不能出现 "INFP" "ENTJ" 等类型
- 不能把角色选择映射到 MBTI 维度
- 不能暗示产品是人格测试

### 2. 禁止心理诊断

- 不能出现 "心理诊断" "心理评估" "心理健康测评"
- 不能使用 "你可能有XX倾向" 等诊断话术
- 不能把剧情选择等同于心理状态

### 3. 禁止医疗术语

- 不能出现 "抑郁症" "焦虑症" "焦虑障碍" "人格障碍"
- 不能暗示剧情选择与心理健康相关

### 4. 禁止"真实人格"

- 不能出现 "真实人格" "真实自我" "你的人格是"
- 不能把剧情选择等同于真实人格判定

### 5. 禁止改写原剧

- 不能改写原剧结局
- 不能替角色生成新的正片台词
- 不能声称 AI 生成的内容是原剧内容

### 6. 禁止无审核发布

- 所有内容包必须 needs_human_review: true
- 运营审核后才能发布到用户端

## 必须包含

### 1. 人工审核标记

- needs_human_review 必须为 true
- agent_meta.human_review_required 必须为 true

### 2. 剧透标记

- 第3、4、5、6集相关内容必须有 spoiler_flags
- node_003: medium
- node_004: high
- node_005: high

### 3. 切片候选声明

- 所有切片标注必须叫 "切片候选"
- 不能叫 "真实命中片段"
- 不能声称已经真实解析视频

### 4. AI 生成声明

- agent_meta.note 必须说明是 AI 生成初稿
- 必须标注需要运营审核

## 产品定位表述

- 授权原剧内容的轻量 AI 互动叙事层
- 用 AI 将剧集关键剧情节点转化为用户可参与的剧情选择题
- 通过原剧切片反馈、角色人格光谱和好友对照
- 将长视频观看从"单向观看"升级为"可选择、可反馈、可分享"的互动体验
- 平台侧 Agent Console 的价值：将互动测试内容生产从"人工逐剧定制"升级为"AI 生成初稿 + 运营审核发布"的规模化生产流程
