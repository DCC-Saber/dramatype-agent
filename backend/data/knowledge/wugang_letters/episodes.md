# 雾港来信 分集概要

## 第1集：来信

- **时长**：45分钟
- **核心事件**：一封没有署名的来信寄到四位故人手中
- **关键场景**：走廊转角发现朋友藏信（00:12:10-00:14:30）
- **冲突类型**：信任与怀疑
- **互动节点**：被藏起的来信
- **剧透级别**：看前无剧透
