# 剧集基本信息

剧名：雾港来信

类型：悬疑 / 群像 / 情感抉择 / 6集短剧

剧情简介：
十年前，雾港发生一起未解旧案。十年后，一封没有署名的来信寄到四位故人手中。每个人都知道一部分真相，也都在隐瞒一部分过去。随着调查深入，他们必须在保护关系、追求真相、遵守规则和打破僵局之间做出选择。

# 角色设定

## 林澈

角色原型：冷静观察者

关键词：克制、观察、延迟判断、证据优先

角色说明：
他不急于摊牌，总是在别人忽略的细节里寻找真相。

行动逻辑：
先观察，再判断；先找证据，再行动。

## 许知夏

角色原型：共情守护者

关键词：关系、保护、共情、承担

角色说明：
她总是先看见人的伤口，再判断事情的对错。

行动逻辑：
优先保护关系和无辜者，再追问真相。

## 周野

角色原型：冒险破局者

关键词：行动、冲动、突破、直觉

角色说明：
他不喜欢等待答案，会直接把门撞开。

行动逻辑：
先行动打破僵局，再在变化中寻找答案。

## 顾沉

角色原型：秩序裁决者

关键词：规则、边界、公正、理性

角色说明：
他相信边界，因为没有边界的善意也会伤人。

行动逻辑：
用规则和边界控制风险，优先维护公正。

# 关键剧情节点

## 节点1：被藏起的来信

集数：第1集

时间码：00:12:10-00:14:30

剧情：
主角发现朋友偷偷藏起了一封关键来信，而这封信可能和十年前的旧案有关。朋友没有发现自己被看见。

冲突类型：
信任与怀疑

互动价值：
用户可以代入“发现朋友隐瞒真相后要不要立刻追问”的困境，不同选择能体现不同角色的行动倾向。

## 节点2：信任的人背叛你

集数：第2集

时间码：00:18:40-00:22:15

剧情：
主角发现最信任的人对自己撒了谎，但这个人曾经在主角最危险的时候救过主角。

冲突类型：
关系保护与真相追问

互动价值：
用户需要在情感信任和事实怀疑之间做选择。

## 节点3：救人还是保住证据

集数：第3集

时间码：00:27:00-00:31:20

剧情：
主角只有10分钟。一个人被困在码头仓库里，而能证明真相的录音也可能被销毁。

冲突类型：
生命优先与证据优先

互动价值：
用户需要在即时救援和长期正义之间做选择。

## 节点4：是否公开真相

集数：第5集

时间码：00:35:10-00:39:00

剧情：
主角掌握了足以推翻所有人关系的真相。但一旦公开，一个无辜的人也会被舆论伤害。

冲突类型：
公开真相与保护无辜者

互动价值：
用户需要判断真相是否应该被立即公开。

## 节点5：最后的选择

集数：第6集

时间码：00:41:30-00:45:00

剧情：
真相即将揭开，但主角发现自己也曾间接造成了这场悲剧。所有人都在等主角开口。

冲突类型：
自我承担与完成目标

互动价值：
用户需要在承担责任和继续行动之间做最终选择。

# 安全边界

2. 不把结果包装成 MBTI。
3. 不改写原剧结局。
4. 不替角色生成新的正片台词。
